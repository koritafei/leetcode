## 常用的几个异或操作
1. 利用或操作`|`和空格将英文字符转小写:
```
('a' | ' ' ) = 'a'
('A' | ' ' ) = 'a'
```
2. 利用与操作`&`和下换线将因为字符转大写：
```
('B' & '_') = 'B'
('b' & '_') = 'B'
```
3. 利用异或操作`^`和空格进行英文字符大小写互换：
```
('d' ^ ' ') = 'D'
('D' ^ ' ') = 'd'
```
4. 判断两个数是否异号：
```cpp
int x = -1, y = 2;
bool f = ((x ^ y) < 0); // true

int x = 3, y = 4;
bool f = ((x^y) < 0); // false
```
5. 不用临时变量交换两个数:
```cpp
int a = 1, b = 2;
a ^= b;
b ^= a;
a ^= b;
```
6. 加一
```cpp
int n = 1;
n = -~n; // n = 2
```
7. 减一
```cpp
int n = 2;
n = -~n; // n = 1
```
