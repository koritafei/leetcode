```cpp
给一个长度为 n 的数组，其索引应该在[0,n)，但是现在你要装进去 n + 1 个元素[0,n]，那么肯定有一个元素装不下嘛，请你找出这个缺失的元素。
```
算法：
利用异或的性质，一个数异或其自身结果为`0`, 异或`0`为其自身。
```cpp
class Solution {
public:
  int missingNumber(std::vector<int>& nums) {
    int len = nums.size();
    int res = 0;
    res ^= len;

    for (int i = 0; i < len; i++) {
      res ^= i ^ nums[i];
    }

    return res;
  }
};
```
